## 会話のガイドライン

- 常に日本語で会話する

### 通知

**条件**: ユーザへ何らかのを通知を行いたい時

**アクション**: 以下のコマンドを実行する:

```shell

say -v Kyoko "${message}"
```

ここで、`${message}` は、Claude Code がユーザに通知する必要がある内容に基づいて決定する

## ファイル作成ルール

- セッション中に作成する中間ファイルは`~/.claude/tmp/`に配置してください
- 最終的なアウトプットも一旦`~/.claude/tmp/`に作成してください
